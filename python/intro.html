Introduction to Python

Python is a programming language created around 1991, and it has became increasingly popular recently. We'll be using Python 3, which was released in 2008. Python's official website includes installation options and the documentation.

A program is a sequence of instructions for the computer to follow. print(), one of the most fundamental "instructions", allows the computer to communicate with you by printing (displaying) something to the screen.

The "hello, world!" program written in Python 3 is:

print('hello, world!')

The outputs produced when the program is executed are:

hello, world!

The print() function will output whatever you put in the parentheses, which is "hello, world!" in this case. The text "hello, world!" is put in single-quotes to denote that it is a string (human text) and not more computer code. Here are some more examples of print():

print('Python is not actually named after the reptile, but a British comedy group called the Monty Python.')
print()
print(3.14)
print('The year is 2020 and everyone is stuck at home!')

The outputs:

Python is not actually named after the reptile, but a British comedy group called the Monty Python.

3.14
The year is 2020 and everyone is stuck at home!

Some key takeaways so far:

  • The computer executes code line by line in the order that they appear in the program (you can manipulate this with "control flow", which we'll discuss later)
  • Strings (human text) are enclosed in single- or double-quotes to differentiate them from code, while numbers like 3.14 and 2 are not
  • print() outputs everything you put between the parentheses, which can be strings or numbers
  • You can even give print() multiple things to output by separating them with commas like print('pi =', 3.1415926, 'approximately'). Python will print each of them in order with spaces in between. This would give the same results as print('pi = 3.1415926 approximately')

Try: use 3 print()s to output "Python says hi!", "e is roughly 2.718", and "qwerty" on three different lines.

Errors

Programs don't always work. Sometimes our code violates rules of the Python language, and it stops running. This code snippet contains some errors:

print('To be, or not to be, that is the question:")
print("Whether 'tis nobler in the mind to suffer...")
print()
print(-- Hamlet (Act 3, Scene 1))

Python will usually tell you what type of error occurred and on which line it thinks the error is. However, the error messages are not always beginner-friendly. It takes some practice to become good at spotting these errors.

Try: run the code above and take a look at the error messages, then fix them.

Math

We've mentioned the two most important types of data that the computer works with: numbers and strings. Let's get into some more details.

  • Examples of strings: "You can include 'single-quotes' in a double-quote string!", 'Strings can include numbers in them 123454321', '1024' (this is still a string and not a number), '' (empty string)

You can also use extra parentheses to group things and specify the order of operations: print((3+5) * (6+7)), print(10 * (6 - (2+3))).

To calculate exponents, use ** (e.g., 5**2).

There's one more math operator that programmers often use: %. It calculates the remainder when the two numbers are divided. For instance, 5%3 is equal to 2 because 5 divided by 3 has remainder 2. 26%10 is equal to 6, and 100%2 is equal to 0.

Variables

You can tell Python to store some values that you may want to use for later with variables. A variable is like a storage box in the computer memory in which you can put things, including strings and numbers. You can change the value of the variable over time as necessary.

# year is a variable, and we're assigning the value 2020 to it
year = 2020
print(year)
2020

A variable is created when you give it a value (like putting something into the storage box). In this example, we're giving the variable year the value 2020, and for the rest of this program, Python will remember that year has the value 2020. Therefore, printing the variable year outputs 2020. Note that we don't put quotations around year when printing it, because if we did Python would treat it as a string and print the word "year" instead.

You can give your variables any name using letters, numbers, and underscores (no spaces and must start with a letter or underscore) as long as it doesn't conflict with reserved keywords. It is usually helpful to name your variables with something that helps you remember what the variable is storing. Also, note that Python variable names are case-sensitive.

description = 'sunset'
hour = 6
minute = 52

print("It's", description, "right now, and the time is", hour, ":", minute)
It's sunset right now, and the time is 6 : 52

In this example above, we gave print() six things to output, and print() conveniently inserts spaces between them for us. Because description is a string, we could also write print("It's " + description + " right now..."), but we would have to add a space after "It's" and before "right".

Let's take a look at a slightly more complex example:

a = 3
b = 4
print(a + b)

a = 5
b = 7
print(a + b)

a = 3
c = a + b
print(c)
print(c * c)

As you see, you can change the value of a variable simply by assigning a new value to it.
