Inputs and If Statements

A brief note regarding functions: print(), int(), float(), and str() are all examples of functions (notice how all of them are followed by a set of parentheses). We'll delve into the details of functions (and how to create our own) later on, but for now think of functions as "mini programs" that perform specific tasks for us. We could give the int() function a decimal like 5.8, and it will round it down to the nearest integer and give us back 5. Read this diagram by following the arrows, starting at the top.

                                what we feed into the function
                                              |
                                              v
what the function gives back to us: 5 <- int(5.8)

What we've illustrated here is that we feed data into a function (and put it in the parentheses), the function processes it, and then gives something back to us (called the return value). In this case, we fed 5.8 into the function int(), and the return value was 5.

Perhaps the max() function illustrates this point better. max() takes in a number of values (separated by commas) and returns the largest of them. The arrows again indicate what we feed into the function and what it returns to us.

              |                                  |
              v                                  v
14 <- max(10, 8, 14)               240 <- max(240, 20)

In print(max(10, 8, 14)), we've nested the print() and max() functions. The return value of the max() function becomes what we feed into print() because it's enclosed in the set of parentheses corresponding to print().

              | we feed 10, 8, and 14 into max()
              v
print(max(10, 8, 14))
             |
      *------*  max(10, 8, 14) evaluates to 14;
      |         14 is fed into print()
      v
print(14)

Inputs

The "opposite" to the print() function is the input() function. Instead of displaying something to you, it allows you to type something into the computer. input() returns what the user typed as a string, and you can then assign that to a variable to use it later.

print('Type something below:')
message = input()
print('You said', message)
Type something below:
the sky is blue
You said the sky is blue

We colored the inputs we typed in light gray. After we press enter (pressing enter tells Python that we're done typing) to send the inputs through, the variable message would be assigned the value of 'the sky is blue'.

One thing to notice is that the input() function always returns (gives you) a string, even if the actual inputs were meant as numbers. For instance, if we typed in the number 5, input() would give you '5' instead. To address this, we need to use int() to convert our inputs to numbers.

print('Enter a number:')
a = int(input()) # gets some input, and convert it to a number
print('Enter another number:')
b = int(input())
print(a + b)
Enter a number:
12
Enter another number:
15
27

If Statements

You can create programs that make decisions using if statements. Often, you'll want your program to behave differently in different situations (perhaps based on the inputs of the user). An if statement allows you to do exactly that. It runs a block of code only if a certain condition is met:

print('Enter your age:')
age = int(input())
if age >= 15:
    print('You can learn to drive and get a permit.')
    print('At age 18, you will be able to get a full license.')
print('...')
Enter your age:
17
You can learn to drive and get a permit.
At age 18, you will be able to get a full license.
...

Here, Python checks if the value of age is greater than or equal to 15. If it is, then the block of code that is indented will be executed. Otherwise, it ignores the indented block of code and goes straight to print('...'). The dots are always printed out regardless of the value of age because it is not part of the indented block.

These are the parts of a typical if statement:

  • The keyword "if"
  • A condition, often a logical comparison using
    < (less than),
    > (greater than),
    <= (less than or equal to),
    >= (greater than or equal to),
    == (equal to),
    or != (not equal to)
  • The colon (:)
  • An indented block of code

A typical mistake is using "=" when "==" should have been used. Remember that "=" means variable assignment, while "==" means comparison. Also, because indentation is important in Python, make sure that your code within the if block is properly aligned.

Optionally, you can add multiple elifs (else if) and an else to go with an if statement. elif allows you to test one or more other conditions, and else provides an "otherwise" case if none of the tests above evaluated to True.

print('Would you like to take the subway or the bus?')
transportation = input()
if transportation == 'subway':
    print('Take line 13, then transfer to line 5.')
elif transportation == 'bus':
    print('Take bus 392.')
Would you like to take the subway or the bus?
bus
Take bus 392.

An elif clause is only tested if all the elifs and if above it tested to False (hence the name else if). As soon as one of the clauses tests to True, its corresponding block of code is executed, and all the following elifs and else are ignored. In other words, only one if/elif/else block will be executed.

print('Enter a number:')
a = int(input())
print('Enter another number:')
b = int(input())

if a < b:
    print(a, 'is less than', b)
elif a > b:
    print(a, 'is greater than', b)
else:
    print('They are equal!')
Enter a number:
6
Enter another number:
6
They are equal!

You don't specify a condition for else because it's the "otherwise" case.

The code above could be summarized in this diagram:

# getting inputs...
        |
        v       False                 False
    if a < b:  ------>  elif a > b:  ------> else:
        |                    |                 |
        | True               | True            |
        v                    v                 v
  print('Less')       print('Greater')    print('Equal')
        |                    |                 |
        |--------------------*-----------------*
        |
	v

Let's take a look at one last (more complicated) example.

print('Quiz time! Enter your name:')
name = input()
num_correct = 0 # how many questions you answered correctly

if name == 'secret answer key':
    print('*** The answers are 5, 26, and 8 ***')
else:
    # Question 1
    print('155 / 31 = ')
    answer = int(input())
    if answer == 5:
        print('Correct!')
        num_correct = num_correct + 1
    else:
        print('The correct answer was 5.')

    # Question 2
    print('How many letters are there in the English alphabet?')
    answer = int(input())
    if answer == 26:
        print('Correct!')
        num_correct = num_correct + 1
    else:
        print('The correct answer was 26.')

    # Question 3
    print('How many planets are there in the Solar System?')
    answer = int(input())
    if answer == 8:
        print('Correct!')
        num_correct = num_correct + 1
    elif answer == 9:
        print('Very close, but not quite!')
    else:
        print('The correct answer was 8.')

    print(name + ', you got ' + str(num_correct) + ' out of 3 points.')

When you have if statements within other if statements like in this example above, make sure that your indentation levels are consistent.

Finally, you can use the and, or, and not operators to combine multiple conditions in a single if statement. For instance, the following code only prints "Sad day" if both the conditions are met.

weather = input()
day = input()
if weather == 'rainy' and day == 'Monday':
    print('Sad day')
