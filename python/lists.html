Lists

Lists allow you to work with a collection of items and easily store a large amount of data. Let's say we want to represent the temperatures in a week. Rather than creating 7 separate variables for each of the temperatures, we make a list to contain all 7 of them. A list is enclosed by square brackets, and the items of the list are separated by commas.

temperatures = [33, 32, 32, 34, 35, 32, 31, 31]
print(temperatures)
[33, 32, 32, 34, 35, 32, 31, 31]

You can similarly make a list of strings, as in weather = ['cloudy', 'rainy', 'cloudy', 'sunny', 'sunny', 'rainy', 'thunderstorms']. You can also make an emtpy list using a pair of brackets with nothing between them.

Using .append(), you can grow the list by adding elements to the end of the list.

fruits = ['watermelons', 'kiwis', 'cherries']
fruits.append('blueberries')
print(fruits)
fruits.append('pears')
fruits.append('strawberries')
print(fruits)
['watermelons', 'kiwis', 'cherries', 'blueberries']
['watermelons', 'kiwis', 'cherries', 'blueberries', 'pears', 'strawberries']

You can access specific elements from a list by using its index, or position in the list. Index numbers start at 0 in Python, which means the very first element in a list is index 0. If the list is fruits = ['watermelons', 'kiwis', 'cherries'], then fruits[0] refers to 'watermelons', fruits[1] refers to 'kiwis', and fruits[2] refers to 'cherries'. Note how element [2] is the last element in a list with 3 things in it (because the first one is [0]).

If you try to access an element that is outside of a list's bounds (e.g., if the list has 5 elements and you try to grab index [10]), Python will give an error.

fruits = ['watermelons', 'kiwis', 'cherries']
print(fruits[1])
fruits[1] = 'bananas'
print(fruits)
kiwis
['watermelons', 'bananas', 'cherries']

Slicing

Python also allows you to get a particular part of a list, described by a starting index and an ending index. "Slicing" a list gives you a new list containing only the specified part.

fruits = ['watermelons', 'kiwis', 'cherries', 'blueberries', 'pears', 'strawberries']
some_fruits = fruits[1:4]
print(some_fruits)
['kiwis', 'cherries', 'blueberries']

In the example above, the slice starts at index 1 ("kiwis") and ends right before index 4 ("pears"), and contains 4 - 1 = 3 elements (specifically, positions 1, 2, and 3). Note that the starting index is included in the slice while the ending index is excluded. As another example, fruits[2:6] would contain 6 - 2 = 4 elements (2, 3, 4, and 5), from "cherries" to "strawberries".

You can omit the starting or ending index (or even both). If the start is omitted, it is assumed to be 0 (slice from the beginning of the list). If the end is omitted, it is assumed to be the length of the list (slice until the end). Thus, if both are omitted, the slice contains the entire list, essentially making a copy of the original.

Use the len() function to find the length of a list. The same function also works on strings (in which case it returns the number of characters in the string).

fruits = ['watermelons', 'kiwis', 'cherries', 'blueberries', 'pears', 'strawberries']
print(len(fruits))
print(len('hello there'))

Lists and Loops

Loops naturally go with lists, allowing you to repeat an operation on all the elements. We may need to loop over an entire list to do something to each of the things in it. Both for and while loops can be used to do this, but a for loop is usually more convenient.

The code below shows the two most typical ways to loop through a list.

colors = ['yellow', 'orange', 'red', 'violet', 'blue', 'green']
print('Method 1:')      
for x in colors:
    print(x)
print('Method 2:')
for i in range(len(colors)):
    print(colors[i])

Let's first look at the first method. Recall that a for loop follows the format of "for item in collection_of_items". Here, because there are 6 elements in the list colors, the loop repeats for 6 times. Each time through the loop, the variable x takes on the value of the next element in the list.

In method 2, you can pretend the loop is "for i in [0, 1, 2, 3, 4, 5]" because len(colors) is 6, and range(6) gives you the numbers 0 through 5. Thus, the first time through the loop, i is 0, and printing colors[i] gives colors[0] ("yellow"). The second time through, i is 1, therefore colors[1] is printed out, and so on.

The next program finds the largest number in the list nums (achieving the effect of max(nums) essentially).

nums = [3, 5, 2, 4, 9, 7, 1]
largest = nums[0]
for n in nums:
    if n > largest:
        largest = n
print(largest)
9

It is also very common to use a for loop with range() to iterate through a list. Let's modify the program above so we also print out what index the maximum element is located at.

nums = [3, 5, 2, 4, 9, 7, 1]
largest = nums[0]
largest_pos = 0
for i in range(len(nums)):
    if nums[i] > largest:
        largest = nums[i]
        largest_pos = i
print(largest, 'at index', largest_pos)
9 at index 4

This final example performs bubble sort on a list of numbers to organize them in ascending order. The idea is that you look at each element in the list and compare that element to all the elements that come after it. If the one that comes after is smaller, these two are out of order and should be swapped.

nums = [3, 5, 2, 4, 9, 7, 1]
for i in range(len(nums)):
    for j in range(i+1, len(nums)):
        if nums[j] < nums[i]:
            nums[i], nums[j] = nums[j], nums[i]
print(nums)
[1, 2, 3, 4, 5, 7, 9]

On line 3, we gave range() two numbers instead of the usual one. Like list slicing, range() with two numbers specify where to start and stop. In this example, we start at one after whatever value i currently is and stop right before 7 (the length of the list). Therefore, if i is currently 3, then j takes on the values 4, 5, and 6 (referring to all the elements that come after num[3]).
