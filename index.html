Abacles (n.): a word I invented to refer to the beads on an abacus, the calculating tool. To pronounce this word, combine "apple", "baton", and "icicles."

This website is currently home to the Panther Creek Programming Team. You can also find a handy ASCII table, and a tool that labels pinyin for Chinese ebooks.
